# 氣功 · Application Qi Gong Quotidien

Application mobile web pour la pratique quotidienne du Qi Gong, basée sur la Médecine Traditionnelle Chinoise (MTC).

## Fonctionnement

1. Choisissez un **moment** de la journée (Matin, Midi, Soir, Automassage, Programme Complet)
2. Sélectionnez une **pratique** dans le menu déroulant selon votre focus (saison, organe, zone corporelle)
3. Consultez la **fiche descriptive** (durée, focus, organes ciblés, niveau)
4. Lancez la vidéo directement dans l'application

## Catégories disponibles

| Moment | Vidéos | Focus |
|--------|--------|-------|
| 🌅 Matin | 5 | Éveil Yang · Immunité · Respiration |
| ☀️ Midi | 12 | Saisons · Méridiens · Maître Ke Wen |
| 🌙 Soir | 2 | Détente · Sommeil profond |
| 💆 Automassage | 3 | Visage · Acupression · Stress |
| ⏱ Complet | 3 | Corps entier · 35–60 min |

## Utilisation

Ouvrez simplement `index.html` dans votre navigateur — aucune installation requise.

L'application est optimisée pour mobile (iOS & Android).

## Sources

Toutes les vidéos sont issues de YouTube. Pratiques issues de la MTC et du Qi Gong traditionnel.
